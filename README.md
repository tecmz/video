# 魔众视频管理系统

国内领先的技术团队开发商，是搭建视频系统最佳选择的程序源码，支持多端同步浏览。

## 运行环境


```
操作系统
Linux/Unix 或&nbsp;&nbsp;Windows
软件环境
Laravel 5.1的运行环境
Apache/Nginx , PHP 5.5.9+ / PHP 7.0 , MySQL 5.0+
```
